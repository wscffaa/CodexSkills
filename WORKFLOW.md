# Workflow First

这个仓库现在的主主题是 **Workflow Stack**，不是“所有 skills 的平铺目录”。

如果你是第一次进入仓库，先读：

1. `Codex/workflow/README.md`
2. `Claude/workflow/README.md`
3. 根目录 `README.md`

## Core Idea

默认工作流主线是：

- `unified-workflow`
- `cybernetic-systems-engineering`
- `harness`
- `harness-engineering`
- `interview-to-plan`
- `plan-to-issues`
- `codex-csv-loop`

这条主线解决：

- 非平凡开发
- 复杂工程任务
- 长时、可恢复执行
- PRD / CSV 驱动的任务管理

其它 skill 继续保留，但默认归为 optional。
